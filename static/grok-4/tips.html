Grok-4 Tips

  • API Access: Grok-4 uses an OpenAI-compatible API available at https://api.x.ai/v1
  • Model Name: Use grok-2-image for image generation requests
  • Authentication: Requires an X.AI API key set in your environment
  • Response Format: Supports b64_json or url response formats
  • Rate Limits: Be mindful of rate limiting when generating multiple images
