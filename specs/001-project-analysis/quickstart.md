# Quickstart: Project Analysis

This guide provides instructions on how to use the Project Analysis tool.

## Prerequisites

- Deno must be installed on your system.

## Usage

1. **Run the analysis**:

   Navigate to the root of the project and run the following command:

   ```bash
   deno run --allow-read --allow-write src/cli/main.ts
   ```

2. **View the output**:

   The analysis will be saved in a file named `project-analysis.md` in the root of the project.
