Tips

Style reference - sref and sw

--sref [image url]

You can generate using a style found in a known image by adding --sref [URL]. This will make the generation look like it was in the same style as the specified image. 

--sw [weight]

You can control how much the reffered image should influance the generation using --sw [0-100]
