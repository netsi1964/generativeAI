1) Trying to generate inspired by a certain artist

If you are looking to generate images inspired by a certain artist, you can sometimes add simply informaion about the artist in the prompt. It is not always working however. Midjourney has the ability to generate images based on an existing image using the style reference parameter: --sref [image url]. Try reading more here: Midjourney Parameter List or see my examples in the Midjourney service page.
